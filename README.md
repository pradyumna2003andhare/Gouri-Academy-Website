# Gouri Academy

Gouri Academy is a modern educational web platform designed to help students access study materials, results, and academic resources in one place. The project focuses on creating a clean, responsive, and user-friendly interface for students and educators.

---

## Features

- Student-friendly responsive UI
- Study material management
- Student results section
- Clean dashboard design
- Easy navigation system
- Mobile responsive layout

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap
- EJS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/gouri-academy.git
```

Move into the project folder:

```bash
cd gouri-academy
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```
